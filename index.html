Python学习笔记(6)

Python 术语

  • “self” —— 这是一个方法参数, 总是指向当前对象实例

Points

  • 可以在类中定义init()方法来初始化对象实例
  • 类中定义的每个方法都必须提供self作为第一个参数

Python学习笔记(5)

Python 术语

  • 原地排序 —— 转换然后替换 —— sort()
  • 复制排序 —— 转换然后返回 —— sorted()
  • 列表推导 —— 在一行上指定一个转换(而不是使用迭代)
  • 分片 —— 从一个列表访问多个列表项
  • 集合 —— 一组无序的数据项, 其中不包含重复项

Points

  • 向sort() 或 sorted() 传入reverse = True 可以按降序排列数据
  • 如果有以下代码
    newList = []
    for t in oldList :
    newList.append(len(t))
    使用列表推导重写这个代码, 可以写作:
    newList = [len(t) for t in oldList]
  • 要访问一个列表中的多个数据项, 可以使用切片. 例如:
    my_list[3:6] (这会访问列表中从索引位置3直到(但不包括)索引位置6的列表项)
  • 使用set() 工厂方法可以创建一个集合

Python学习笔记(4)

Python 术语

  • “不可变类型”—— Python中的一些数据类型一旦赋值, 这个值就不能再改变
  • “腌制” —— 将数据对象保存到一个持久储存中的过程
  • “解除腌制” —— 从持久储存中恢复一个已保存的数据对象的过程

Points

  • str() BIF可以用来访问任何数据对象(支持串转换)的串表示
  • with 语句会自动处理所有已打开文件的关闭工作, 即使出现异常也不例外. with语句也使用as 关键字
  • 标准库的pickle模块允许你容易而高效滴将Python数据对象保存到磁盘以及从磁盘中恢复
  • pickle.dump() 函数将数据保存到磁盘
  • pickle.load() 函数从磁盘恢复数据

Python学习笔记(3)

Python 术语

  • “异常”(exception)因运行时错误而出现, 会产生一个traceback
  • “traceback”是出现的运行时错误的一个详细的描述

Points

  • 使用open() BIF打开一个磁盘文件, 创建一个迭代器从文件读取数据, 一次读取一个数据行
  • readline()方法从一个打开的文件读取一个数据行
  • seek()方法可以用来将文件”退回”到起始位置
  • close()方法关闭一个之前打开的文件
  • split()方法可以将一个字符串分解为一个子串列表
  • Python中不可改变的常量列表成为元组(tuple). 一旦列表数据赋至一个元组, 就不能再改变. 元组是不可改变的
  • 数据出现不期望的格式时出现ValueError
  • 数据无法正常访问时出现IOError(例如, 可能你的数据文件已经移走或者重命名)
  • find()方法会在一个字符串中查找一个特定的子串
  • not关键字将一个特定的条件取反
  • try/except 语句提供一个异常处理机制, 从而保护可能导致运行时错误的某些代码行
  • pass语句就是Python的空语句或null语句, 它什么也不做

Python学习笔记(2)

Python 术语

  • 使用”三重引号字符串”可以在代码中加入一个多行注释
  • 使用# 代表单行注释
  • Python内存中的名字就存放在”命名空间”中
  • Python的主命名空间名为main

Points

  • 模块是一个包含Python代码的文本文件
  • setup.py 程序提供了模块的元数据, 用来构建,安装和上传打包的发布
  • 使用import语句可以将模块导入到其他程序中
  • Python中的各个模块提供了自己的命名空间, 使用module.function()形式调用模块的函数时, 要用命名空间名限定函数
  • 使用import语句的from module import function形式可以从一个模块将函数专门导入到当前命名空间
  • 包含end=‘’ 作为print() BIF的一个参数会关闭其默认行为(即在输入中自动包含换行)
  • 如果函数参数提供一个缺省值, 这个函数参数就是可选的

Python学习笔记(1)

Python 术语

  • BIF ——内置函数
  • 组(suite) —— Python代码块, 会通过缩进来指示分组

Points

  • 标识符是指示数据对象的名字, 标识符没有”类型”, 不过标识符所指示的数据对象有类型
  • 列表是一个数据集合, 数据项之间用逗号分隔, 整个列表用中括号包围
  • 列表就像是打了激素的数组, 并不完全等于数组
  • 列表可以存放任意数据类型, 而且数据可以是混合类型. 列表还可以包含其他列表
  • 列表可以随需要伸缩, 数据使用的所有内存都由Python为你管理
  • Python使用缩进将语句归组到一起
  • isinstance() BIF会检查一个标识符是否指示某个指定类型的数据对象. 和OC中的isKindOf类似
  • 使用def来定义一个定制函数
